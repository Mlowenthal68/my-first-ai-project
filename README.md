# My First AI Project

A Python data analysis script built with pandas, numpy, and requests.

## Requirements

- Python 3.12+
- virtualenv

## Setup

```bash
# Create and activate virtual environment
virtualenv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python analysis.py
```

## What it does

- Builds a sample sales dataset using **pandas**
- Computes revenue, summary statistics, and category breakdowns
- Uses **numpy** for mean, median, and standard deviation on revenue
- Fetches sample external data using **requests**
